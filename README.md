# KDE-Window-Maximize-Minimize-Shortcuts
Add shortcuts to KDE for maximize (not toggle) and a shortcut to either unmaximize or minimize depending on window state.

Installation:
Place WindowMaxMin folder into /usr/share/kwin/scripts
Enable in Settings > Window Management > Kwin Scripts
Set shortcuts in Settings > Keyboard > Shortcuts ("Maximize Window (Not Toggle)" and "Unmaximize Window / Minimize")

Optional:
I use keyd to map Mouse Forward + Scroll up to be F18, which I use for "Maximize Window (Not Toggle)"
I use keyd to map Mouse Forward + Scroll down to be F17, which I use for "Unmaximize Window / Minimize"
